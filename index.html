Configuración de base de datos compartida
Esta app usa Firebase Firestore para sincronizar los datos en tiempo real entre todos los usuarios. Ingresá las credenciales de tu proyecto Firebase para conectar. Solo necesitás hacerlo una vez — quedan guardadas en este navegador.
Cómo obtener las credenciales
1
Entrá a console.firebase.google.com → Crear proyecto
2
En el proyecto → Firestore Database → Crear base de datos → Modo de prueba
3
Configuración del proyecto (⚙️) → Tus apps → ícono Web </> → Registrar app
4
Copiá los valores del bloque firebaseConfig y pegálos abajo
Los datos se comparten entre todos los que usen esta configuración.
